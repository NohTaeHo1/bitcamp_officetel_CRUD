# 오피스텔 테이블 백엔드

## 소개
오피스텔 테이블 백엔드는 프론트엔드 애플리케이션과 통신하여 오피스텔 매물 데이터를 관리하는 서버 애플리케이션입니다. Spring Boot 3를 활용하여 안정적이고 확장 가능한 REST API를 제공합니다.

## 주요 기능
- **회원 관리**: 사용자의 회원가입, 로그인, 로그아웃 및 탈퇴 기능을 제공합니다.
- **매물 관리**: 오피스텔 매물의 등록, 수정, 삭제 및 조회 기능을 제공합니다.
- **실시간 필터링**: JPQL과 QueryDSL을 활용하여 실시간으로 매물 데이터를 필터링합니다.
- **JWT 인증**: JSON Web Token을 사용한 사용자 인증 및 권한 부여를 처리합니다.

## 기술 스택
- **프레임워크**: Spring Boot 3
- **데이터베이스**: MySQL
- **ORM**: Spring Data JPA
- **쿼리**: JPQL, QueryDSL
- **보안**: JWT
